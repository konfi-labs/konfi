# Категорії

Категорії організують продукти в каталозі каналу. Секція Категорії розміщена на
сторінці Каталог під продуктами.

## Список категорій

- Шукай категорію за назвою.
- Оновлюй список після змін структури каталогу.
- Таблиця показує назву, опис, переклади, дату створення і дії.

## Створення та редагування

- **Додати категорію** створює нову категорію.
- Дії рядка редагують або дублюють категорію.
- Форма містить назву, опис, батьківську категорію і SEO.
- Батьківські категорії формують ієрархію для магазину та адміністрації.

## Переклади

- Колонка перекладів використовується для мовних версій.
- Переклади важливі для багатомовної навігації магазину.
- Автопереклад може працювати у фоні й попереджати про помилки.

## Видалення

- Видаляй лише після перевірки, що жоден продукт не залежить від категорії.
- Спочатку перенеси прив'язані продукти до іншої категорії.
- Для тесту нової структури безпечніше дублювати категорію.
