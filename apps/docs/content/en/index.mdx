# Konfi Admin Docs

Use these guides when running day-to-day admin work in Konfi. The docs are
organized around the main operational areas of the admin app and assume you are
working in the correct tenant and channel.

## Core Guides

- [Orders](./orders) - find, create, update, fulfill, invoice, and ship orders.
- [Customers](./customers) - manage customer records, groups, carts, and notes.
- [Products](./products) - maintain catalog products, pricing, media, and links.
- [Categories](./categories) - organize products and manage category translations.
- [Store settings](./store-settings) - configure checkout, shipping, SEO, content,
  and store automation settings.
- [MCP server](./mcp-server) - connect agent clients to approved Konfi tools,
  inspect available tools, and understand example prompts and outputs.

## Working Model

Konfi admin data is channel-aware. Before changing catalog or store settings,
check the active channel selector in the page header. Some pages also expose
actions only when your role has permission or when the required integration is
configured.
