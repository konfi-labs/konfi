# @konfi/epaka

TypeScript client for the Epaka API, generated using Microsoft Kiota.

## Provenance

The checked-in API description is based on Epaka's public API/OpenAPI surface.
The generated client is included in the public repository to preserve workspace
developer experience and interoperability with provider integrations.

Epaka names, API documentation text, endpoint descriptions, examples, and marks
remain attributable to Epaka and are subject to Epaka's current API and
brand/trademark terms. Inclusion in Konfi does not imply Epaka endorsement or
partnership.

## Regeneration

```bash
pnpm --filter @konfi/epaka generate-sdks
```

redistribution notes.
