# @konfi/allegro

TypeScript client for the Allegro REST API, generated using Microsoft Kiota.

## Provenance

The checked-in API description is sourced from Allegro's public OpenAPI
description at `https://developer.allegro.pl/swagger.yaml`. The generated client
is included in the public repository to preserve workspace developer experience
and interoperability with the admin integration.

Allegro names, API documentation text, endpoint descriptions, examples, and
marks remain attributable to Allegro and are subject to Allegro's current API
rules and brand/trademark terms. Inclusion in Konfi does not imply Allegro
endorsement or partnership.

## Regeneration

```bash
pnpm --filter @konfi/allegro generate-sdks
```

See the root `THIRD_PARTY_NOTICES.md` file for generated-client provenance and
redistribution notes.
